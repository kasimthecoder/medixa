# Medixa

Search, compare, and understand medicines — simple and clear.

Medixa is a platform to explore medicines easily.  
Search medicines, compare options, check side effects, and find alternatives in one place.

## Features

- Search medicines by name or condition
- Compare medicines (price, composition, side effects)
- View detailed information
- Find alternative medicines
- Browse by categories (Pain Relief, Fever, etc.)

## Tech Stack

- Next.js, React, Tailwind CSS
- Node.js, Express
- MongoDB

## Status

🚧 In Progress

## Note

Medixa does not sell medicines.  
Information is for educational purposes only. Consult a doctor before use.
